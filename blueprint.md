
# Project Blueprint: Clock and Alarm Web App

## Overview

This document outlines the plan for creating a web-based clock and alarm application. The application will display the current time and allow users to set and manage multiple alarms.

## Features

*   **Digital Clock:** Display the current time, updating every second.
*   **Alarm:**
    *   Set an alarm for a specific time.
    *   The alarm will sound and a notification will be displayed when the set time is reached.
*   **Modern UI:** A clean and user-friendly interface using modern CSS for styling.

## File Structure

*   `index.html`: The main HTML file containing the structure of the web page.
*   `style.css`: The CSS file for styling the application.
*   `main.js`: The JavaScript file containing the logic for the clock and alarm functionality.
